### Overview

A command line program built in python to scan files in a particular directory and classify them as Virus or not Virus.

### Antivirus

#### Introduction

The Antivirus is composed of 3 Types of files : 

- User Interface                (**UI**)
- Virus Checker Interface       (**VCI**)
- Virus Checkers                (**VCs**)
